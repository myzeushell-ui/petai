# Как клонировать свой голос для PetAI-видео

> Пошагово. Заняло у меня (Дмитрий) ~15 минут. Делается один раз.

## Шаг 1 — Записать сэмпл

**Что читать:** [voice-over.md](voice-over.md) — там 8 коротких абзацев, прочитай весь файл подряд = ~1.5–2 мин. Бонус: будущая озвучка ляжет в ту же эмоциональную палитру.

**Где:** маленькая комната с ковром/шторами (поглощают эхо). **Не** кухня и **не** ванная.

**Чем (по убыванию качества):**
1. USB-микрофон (Blue Yeti, Rode и т.п.)
2. Гарнитура от телефона / AirPods
3. Встроенный микрофон ноутбука с близкого расстояния (15 см)
4. Встроенный микрофон смартфона

**Чем записать:**
- **Телефон:** стандартный «Диктофон» → формат WAV или M4A
- **Компьютер:** [Audacity](https://www.audacityteam.org/) (бесплатно) → File → Export → MP3

**Как:**
- Один дубль. Сбился — начни заново.
- Разговорный тон, как другу. Не диктор.
- Пауза 1–2 сек между предложениями — норм.
- Не пой, не шепчи.
- Длина: минимум 1 мин, оптимум 2–3 мин.

**Сохранить:** `petai/docs/video-assets/audio/voice-sample.mp3`

## Шаг 2 — Аккаунт ElevenLabs

1. https://elevenlabs.io/sign-up — войди через Google
2. Подтверди email
3. Тариф **Starter $5/мес** (можно отменить через месяц)
   - Включает Instant Voice Cloning
   - Лимит 30 000 символов = ~30 мин озвучки

## Шаг 3 — Клонировать голос

1. Слева **Voices** → **Add a New Voice**
2. Выбрать **Instant Voice Cloning** (не Professional)
3. **Name:** `PetAI Dima`
4. **Upload:** `voice-sample.mp3`
5. **Labels:**
   - Language: English
   - Accent: Russian/Slavic
   - Age: young adult
   - Gender: male
   - Use case: narrative
6. ✅ "I confirm this is my voice"
7. **Add Voice** → ждать ~30 сек

## Шаг 4 — Скопировать ключи

**API key:** Profile → API Keys → Create API Key → копируй (`sk_...`)

**Voice ID:** Voices → клик на «PetAI Dima» → URL содержит voice_id

## Шаг 5 — Положить в проект

Создать `petai/.env.local`:

```
ELEVENLABS_API_KEY=sk_твой_ключ
ELEVENLABS_VOICE_ID=твой_voice_id
```

(`.env.local` уже в `.gitignore`, не утечёт в git.)

## Шаг 6 — Сказать мне «голос готов»

Я запущу:
- [generate-voiceover.sh](scripts/generate-voiceover.sh) → 8 mp3 твоим голосом
- [assemble.sh](scripts/assemble.sh) → пересоберу final.mp4

Время: ~2 минуты.

## Если что-то пошло не так

| Проблема | Решение |
|---|---|
| Голос звучит роботично | В Voice Settings: Stability 0.55 → 0.65 |
| Не похоже на меня | Перезапиши сэмпл с лучшим mic'ом, Similarity Boost = 0.95 |
| Глотает слова в длинных фразах | Дописать точки в тексте сценария |
| Не тот акцент | Re-clone, в Labels указать Accent: Russian |
| API возвращает 401 | Проверить ELEVENLABS_API_KEY без лишних кавычек |
| API возвращает 429 | Превышен лимит Starter, проверить usage в Profile |

## Стоимость

- Starter: $5/мес (можно отменить после генерации видео)
- Расход на 8 сцен: ~1 600 символов = 5% месячного лимита
- Полная перезапись × 3 раза: ~15% лимита

Хватает с запасом. Один месяц подписки → готовое видео.
