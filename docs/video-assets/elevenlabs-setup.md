# ElevenLabs Voice Clone — Setup Guide

> Клонируем твой голос и генерируем 8 аудио-сценах для видео PetAI.

## 1. Записать голосовой сэмпл

**Цель:** 1–3 минуты чистой записи твоего голоса на английском.

- Тихая комната, **без эха** (диваны/шторы помогают, плитка/стекло — враг)
- Не дальше 15 см от микрофона телефона/гарнитуры
- **Один дубль**, без склеек
- Естественная интонация, без пения и крика
- Формат: **WAV / MP3**, моно, 22 kHz и выше

**Что читать:** любой английский текст 200–400 слов. Подойдёт текст самого сценария (`voice-over.md`) — это даже лучше, потому что просодика совпадёт.

Сохрани в: `C:\Users\PC\petai\docs\video-assets\audio\voice-sample.mp3`

## 2. Аккаунт ElevenLabs

1. Создать аккаунт: https://elevenlabs.io/sign-up
2. **Plan:** Starter $5/мес (Instant Voice Cloning) — для теста достаточно
3. **API key:** Profile → API Keys → Create. Сохрани в `.env.local` проекта:
   ```
   ELEVENLABS_API_KEY=your_key_here
   ```
   Файл `.env.local` уже в `.gitignore` — не утечёт.

## 3. Клонировать голос (через UI)

1. Voices → **Add a New Voice** → **Instant Voice Cloning**
2. Назови: `PetAI Founder Voice`
3. Загрузи `voice-sample.mp3`
4. Labels: `English`, `narrative`, `young adult`
5. **Save** — получишь `voice_id` (вид: `21m00Tcm4TlvDq8ikWAM`). Скопируй.

Положи `voice_id` в `.env.local`:
```
ELEVENLABS_VOICE_ID=твой_voice_id
```

## 4. Сгенерировать 8 аудио-файлов

Бэш-скрипт ниже прочитает `voice-over.md`, выдернет каждую сцену и сгенерирует MP3.

См. файл: `scripts/generate-voiceover.sh` (создаётся отдельно).

Запуск:
```bash
cd petai
bash docs/video-assets/scripts/generate-voiceover.sh
```

Результат — 8 файлов в `docs/video-assets/audio/scene-XX.mp3`.

## 5. Настройки голоса (Voice Settings)

В UI ElevenLabs или в API:

| Параметр              | Значение | Эффект |
|-----------------------|----------|--------|
| **Stability**         | 0.45     | Чуть нестабильный — добавляет эмоций |
| **Similarity Boost**  | 0.85     | Максимум похожести на оригинал |
| **Style Exaggeration**| 0.30     | Лёгкий "actorly" окрас |
| **Speaker Boost**     | ON       | Чище звук |
| **Model**             | `eleven_multilingual_v2` | Лучшее качество на 2026 |

## 6. Если результат не нравится

- **Звучит роботично** → подними Stability до 0.55
- **Не похоже на тебя** → подними Similarity до 0.95, перезаписать сэмпл с лучшим mic'ом
- **Слишком ровно** → понизь Stability до 0.35
- **Глотает слова** → разбей длинные предложения дополнительными точками в тексте
