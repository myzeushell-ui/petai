## 🎯 Что от Димы нужно ПЕРЕД началом подачи

1. **Заполнить личные данные** в `00-MASTER-KIT.md` (10 мин)
2. **Создать LinkedIn профиль PetAI** если ещё нет (15 мин)
3. **Получить заявление о юр.лице** — пока Atlas в процессе, многие программы принимают "Company being formed" статус
4. **(Опционально) Создать pitch deck** — нужен для ~30% заявок. Если нет — пропускаем те где требуется, идём в остальные.

После заполнения мастер-кита — все ответы автоматически готовы. Просто открываешь файл нужной программы → копируешь → вставляешь.

Делать или ставить на стоп — скажи **"делаем 01"** (Microsoft) и я выдам пошаговый walkthrough как в Stripe Atlas.
