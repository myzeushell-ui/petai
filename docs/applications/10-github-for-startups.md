# #10 — GitHub for Startups

## Краткий бриф
- **Что дают:** GitHub Enterprise бесплатно на 1 год ($21/user/мес value = $250+ за пользователя/год), бесплатные Actions minutes, Copilot Business для команды, advanced security features
- **Equity:** ❌ нет
- **Подача:** **бесплатно** — через **Microsoft for Startups** (см. файл #01), это автоматический бонус
- **URL:** https://github.com/enterprise/startups (но реальный поток — через Microsoft Founders Hub)
- **Срок:** активируется автоматом после Microsoft for Startups approval

## ✅ ЧТО ОТ ТЕБЯ НУЖНО ЛИЧНО
- [ ] **Подать #01 Microsoft for Startups** — GitHub Enterprise придёт автоматически как часть пакета
- [ ] Существующий GitHub account (myzeushell-ui уже есть)
- [ ] Создать GitHub Organization для PetAI (бесплатно): `github.com/petai-inc` или `github.com/petai-app`

## 📝 ГОТОВЫЕ ОТВЕТЫ
Эта программа НЕ требует отдельной подачи если уже подал на Microsoft for Startups (#01). GitHub Enterprise license активируется через Microsoft Founders Hub portal.

### Альтернативный путь (без Microsoft):
Подать напрямую на https://github.com/enterprise/startups с теми же данными что Microsoft:
- **Company name:** `PetAI`
- **Website:** `https://petai-ochre.vercel.app`
- **Stage:** `Pre-seed`
- **Why GitHub Enterprise?** `Advanced security scanning for AI/ML codebase, Copilot Business for 1 founder + planned 2 hires, dedicated audit log for SOC2 prep`
